2023年底的欲望实现了吗

转行做程序员：转了，刚好工作半年

去热带玩：乐呵呵的过国庆假期时，我还没有意识到，这是今年最后一个长途旅游的机会。12月紧急去香港补救了一下

纹身：约稿，在最后一个周末去扎了，扎了整整一天，中午扎到到凌晨1点，中途太难熬，哭着说受不了了，吃了止疼片，补充热量：牛肉芝士汉堡和可乐，之后带好眼罩，搬来电暖风，最后坚持纹完。是幽灵公主主题，和我预想不太一样，我想表现其中人和自然冲突的部分，纹身师给了我一个童话图腾，也可以，很漂亮（好贵啊!!）还想继续纹，甚至设计阶段就说好了留个接口

年度好物

降噪耳机

23年12月购入 Bose qc45 头戴式。确实很软，以前用索尼的一款戴久了下颌会痛，这个戴一整天OK，降噪效果也好，用了之后才发现过往人生一直在忍受嘈杂

24年12月，去纹身那天，丢了用了4年的AirPods😭（两周后 edit 其实在外套口袋里），购入AirPods Pro2，半夜下单清早快递敲门（牛逼），降噪效果跟 Bose 差不多，优点是小巧，缺点是少了头戴式耳机「我听不见你说话」的免责声明

总之降噪耳机应该出生就给每个人内置一个

隐形眼镜

酷视定制年抛散光片 ¥200+，我的散光度数比较高，用散光度数折算成近视度数的方法买的隐形眼镜戴上很不舒服。另外高度近视的框架镜成像有明显变形，而且比较重，我不喜欢戴，所以找这种定制片试一试。拿到观察，它直径比普通隐形片大很多，戴上虹膜周围可以看到一圈淡蓝色，厚度也比较高，意外的是戴上、以及长时间佩戴挺舒适

山地自行车

美利达勇士300ds ¥700+ ，二手车店购入，用来通勤，以其陪我去10-20km以外的地方，期间换过夹器和脚踏。最意外的是很练臀腿，很久没力量训练之后重回健身房，深蹲水平快速恢复以及上涨全得益于它，我喜欢它的红黑配色（图片是23款）

户外雨衣

蓝色领域小码 ¥100+。刚开始想买儿童版，店主让我等他们生产小码，一等就等到11月，拿到发现户外的东西做的细节好多，好精致～拿收纳袋装着天天揣包里，从此踏入户外用品的坑～

手电筒

ACEBEAM E75 高显色版 ¥299，因为自行车灯被偷了买的。这个流明不做假，夜骑安心多了。有天线路检修停电，打开手电发现比房间顶灯像自然光多了……认准日亚版高显色灯珠

户外衣物

Rab Vital hoody，确实很抗风，也很轻便，又因为速干，理论上只需要这么一件……缺点嘛欧版又窄又长。有钱的话还想买他们的软壳

男款速干短袖，迪卡侬，甚至还是外贸原单。穿上这大宽肩～让人心神迷醉。今年几件速干衣物一举破除我对天然织物的迷信

讲一讲今年的生活

工作

工作内容

客户端开发。季度为周期，接需求-详细讨论-方案设计-需求澄清-开发-测试-改问题单。苦笑，996不开玩笑。引入了傻逼领导，此地不可久留，今天同事刚给我说：你还年轻，最好不要一直在这

未来发展方向

Java + React + 英语，全栈，2025.06 开始投简历

学习

12周计划

10月开始尝试，到24年结束刚好12周，很多周计划都没完成，不过我觉得这个理念很好，按我的理解就是敏捷？今年我会继续尝试，设定合理的目标并尽量完成

空中英语教室

看了100+天，听力水平和对英语的兴趣有长足进步，也因此今年准备考雅思和找用英语沟通的工作

人际交往

见了好几次网友，每次都是在动漫星城见面哈哈（）玩得很开心！新的一年，想见更多的人

新的一年，如果能找到女朋友，就更好啦，如果真的发生了，那简直像金苹果砸在头上一样，眼冒金星，头晕目眩，露出晕乎乎的笑容

美食

大头虾：法包蘸咖喱牛腩，快吃成 comfort food了

塔高：巴斯克很好吃

越芽越南餐室：朴实的好吃

利苑：酥皮点心和杨枝甘露很——好吃，几个招牌反而觉得一般，菜单上好多还没吃过，继续

脆柿：妈妈寄给我一箱，从懒得拆包裹到迷恋，只有咬一口的距离，估计我吃了10+kg

冰淇淋团体奖：吃了很多巧克力冰淇淋，哈根达斯，GODIVA，KitKat，梦龙有一个黑巧克力口味的，苦苦的很好吃，要是冰淇淋里的牛奶巧克力都换成黑巧克力就好了

运动

得益于骑行，臀腿力量没怎么掉，而且突破记录，最近一次是深蹲 60kg x5 x3，所以我也很想试试攀岩来强化背部肌肉呢

发现我单侧下颌弹响是偏头which means胸椎侧旋、骨盆旋转引起的

其他碎碎念

香港印象：还是小时候看的港片里那个样子呀，英雄本色etc. 城市很有韵味，没有难看的标语，中英文标识很多，我心说生活在这里英语能不好吗，维港那边停了很多帆船，夕阳下很美。我们要在没有黑暗电动车的地方相见。

2025 我要

我要真的创造点什么，哪怕很烂，我要和人讲话，要参与到这恶俗、肤浅、被裹挟、和美丽交织的真实生活里去。欲望代表着生机，渴望才会活着。有时候我觉得自己漂浮在一个灰色的空间里，快乐来过然后流走，负面的体验倒是沉淀在那里，失去勇气、失去热情，冷眼看着，试图不受伤的方法是什么都不做，人是三维的动物，只能选一条路走，我哪一条都不想失去，所以停在路口。我想看风景。

选择意味着部分的放弃，也意味着整体的新生。 人生的困境往往像一个戈尔狄俄斯之结，只有相互纠缠的偏见之团。 抽出你的亚历山大之剑吧！

欲望是好的！我的欲望是什么？

2024 结束时我会来，划掉我满足了的欲望

去年的我说得好👏。那么25年，我要
